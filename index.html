El Técnico es una persona que se dedica a dar respuestas a las necesidades de las personas en el sector Informático. Estos son capaces en desempeñar cualquier rol dentro del Desarrollo de Software.

El Desarrollo de Software es el proceso de analizar, codificar, implementar y crear aplicaciones/programas para Computadoras, Tablet y Celulares basándose en las necesidades del Usuario que las va a utilizar.

¿QUE HACEMOS?

APLICACIONES DE ESCRITORIO: Son las que se encuentran instaladas en los ordenadores como Notebook, Netbooks y PC de Escritorio y no necesitan de internet para funcionar.

APLICACIONES WEB: Son las que se encuentran alojadas en un Servidor Web y se necesita de Internet y un Navegador para poder acceder a ellas.

APLICACIONES PARA DISPOSITIVOS MOVILES: Son las que se encuentran instaladas en teléfonos Inteligentes, Tablet, y otros dispositivos móviles
