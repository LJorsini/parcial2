QUE SIGNIFICA DESARROLLAR SOFTWARE

El Software es el soporte lógico de cualquier dispositivo (computadora o celular) y permiten la realización de tareas específicas, es lo contrario a los componentes físico que son llamados Hardware. La interacción entre el Software y el Hardware hacen que cualquier dispositivo pueda funcionar correctamente ya que el software envía ordenes que el hardware ejecuta.

El Desarrollo de Software es el proceso de analizar, codificar, implementar y crear aplicaciones/programas para Computadoras, Tablet y Celulares basándose en las necesidades del Usuario que las va a utilizar.

Una Aplicación o Programa de Aplicación es un conjunto de instrucciones escritas en algún Lenguaje de Programación y que ejecutadas secuencialmente resuelven un problema específico.
